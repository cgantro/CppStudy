# 오버로드

## [오버로드 코드](./code/overload.cpp)

C++은 함수 이름이 같아도 파라미터 타입이 다르면 다른 함수다.

- overload resolution으로 호출 인자, 후보 함수 집합을 기준으로 가장 적합한 함수를 선택한다.

```cpp
void f(int);
void f(long);

f(10);  // INT
f(10L); // LONG

// 암시적 변환이 여러 개 가능하면 모호해질 수 있음

void f(long);
void f(double);

f(10) // int -> Long, int -> double 모두 가능함
```
