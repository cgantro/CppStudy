## C++ 타입 시스템 정리

#### 타입 = 값의 해석 방식 + 가능한 연산 + 수명/소유권 규칙 + 변환 가능성

1. 어떤 연산이 가능한가
2. 어떤 변환이 안전하게 가능한가
3. 값이 복사되는가, 참조되는가, 이동되는가
4. 객체의 수명과 메모리 해석이 올바른가
5. 템플릿 코드가 어떤 타입에 대해서 인스턴스화 가능한가

```cpp
// 기초 타입

int i;
double d;
bool b;
char c;

// compound type
int* p; // 포인터
int& r = i; // lval 참조
int && rr =10; // rval 참조
int arr[3]; // array
void (*fp)(); // function pointer

// 사용자 정의 타입
struct

enum class
```

### 표현식 타입과 값 범주

표준에서는 모든 표현식이 `lvalue, xvalue, prvalue` 중 하나에 속한다고 한다.

| 값 범주 | 의미                                     | 예시                                |
| ------- | ---------------------------------------- | ----------------------------------- |
| lvalue  | 이름이 있고 식별 가능한 객체             | `x`, `arr[0]`, `*p`                 |
| prvalue | 순수 임시값                              | `10`, `x + y`, `std::string("abc")` |
| xvalue  | 수명을 다해가며 자원을 이동할 수 있는 값 | `std::move(x)`                      |
