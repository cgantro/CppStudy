타입을 매개변수로 받는 컴파일 타임 메커니즘

- 함수 호출에서 템플릿 인자 추론이 함수 템플릿 파라미터 타입과 인자 타입을 비교해 수행된다고 설명함

```cpp
template<typename T>
T maxValue(T a, T b){
    return a > b ? a : b;
}

int main(){
    int a = maxValue(1,2);
    double b = maxValue(1.5,2.5);
}

// 서로 다른 타입을 허용 하려면?
// 반환 타입을 별도로 추론한다.

template<typename T, typename U>
auto add(T a, U b){
    return a + b;
}

auto x = add(1,2.5) // 타입은 double(자동 형 변환)
```

### 템플릿 특수화

```cpp
template <typename T>
struct TypeName{
    static void print(){
        std::cout << "unknown\n";
    }
};

template <>
struct TypeName<int>{
    static void print(){
        std::cout << "int\n";
    }
};

int main(){
    TypeName<double>::print(); // unknown
    TypeName<int>::print(); // int

    // 이렇게 특수 타입만 처리할 수 있다.
}
```

### 타입 특성 검사

```cpp
template <typename T>
void printCategory(){
    if constexpr (std::is_integral_v<T>){
        std::cout << "Integral\n";
    } else if constexpr(std::is_floating_point_v<T>){
        std::cout << "floating point\n";
    }else std:: cout << "other\n";
}

int main() {
    printCategory<int>();
    printCategory<double>();
    printCategory<std::string>();
}
```
